## Docker基本

	容器运用比较广泛，微服务、CI

### 案例

#### 1. 启动Nginx

```
docker run -d -p 80:80 --name nginx 
-v C:/Users/xusj/Git/:/Users/xusj/Git/:rw 
-v C:/Users/xusj/nginx/vhosts/:/etc/nginx/conf.d/:ro  
	registry.cn-hangzhou.aliyuncs.com/xusj_repo/nginx

```
